# AI Practice - April 2026
---

This repository contains implementations of various neural network architectures built in PyTorch from memory, without relying on external resources.

The goal of this project is to strengthen my understanding of core AI/ML concepts by reconstructing models from first principles, including:

- Forward and backward propagation
- Loss functions and optimization
- Model architecture design
- Training loops and debugging

Each folder represents a different model (e.g., MLP, CNN, etc.) implemented step-by-step.

This approach ensures I fully understand how each component works rather than relying on pre-built abstractions.
